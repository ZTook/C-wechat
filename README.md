### 微信公众平台开发Demo(Nodejs + LeanCloud)
使用 LeanEngine ,基于Nodejs的微信公众平台开发项目。


## 文档

1. git clone git@github.com:leancloud/LeanEngine-WechatBot
2. 按照微信公众平台设置修改/routes/wechatbot.js中的相关值
